Get Started

Up and Running in 60 Seconds

Three simple steps to revolutionize your browser automation workflow.

1

Install PSP

Clone the repository and install dependencies with npm or yarn.

# Install PSP globally
npm install -g @psp/cli
# Or clone the repository
git clone https://github.com/samihalawa/PSP-PersistentSessionsProtocol.git
2

Capture Session

Capture your first browser session with one simple command.

# Launch browser and capture session
psp create "Gmail Session"
psp launch --url https://gmail.com
✅ Session captured successfully!
3

Restore Anywhere

Use your captured session in any automation framework or browser.

// In your Playwright/Selenium script
import { Session } from '@psp/core';
const session = await Session.load('Gmail Session');
await session.restore(page);
// ✨ Already logged in!
