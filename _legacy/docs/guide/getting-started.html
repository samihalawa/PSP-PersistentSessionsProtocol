Getting Started with PSP

Learn how to set up and use the Persistent Sessions Protocol in your automation projects. PSP makes browser session management simple and unified across all automation frameworks.

Installation

Option 1: NPM Package (Recommended)

Terminal
# Install core PSP packages
npm install @psp/core @psp/cli
# Install adapters for your framework
npm install @psp/adapter-playwright

Option 2: Clone Repository

Terminal
git clone https://github.com/samihalawa/PSP-PersistentSessionsProtocol.git
cd PSP-PersistentSessionsProtocol
npm install
npm run build

Basic Usage

1. Create a Session

session-example.js
import { Session } from '@psp/core';
import { PlaywrightAdapter } from '@psp/adapter-playwright';
// Create a new session
const session = await Session.create({
  name: 'Gmail Production',
  description: 'Gmail session for automation',
  tags: ['gmail', 'production'],
  storage: 'local'
});

2. Capture Browser State

capture-example.js
// Initialize Playwright and navigate
const browser = await chromium.launch();
const page = await browser.newPage();
await page.goto('https://gmail.com');
// Perform authentication steps...
// Capture the authenticated session
await session.capture(page);
console.log('‚úÖ Session captured successfully!');

3. Restore Session Anywhere

restore-example.js
// Later, in a different script or framework
const session = await Session.load('Gmail Production');
// Create new browser context
const newBrowser = await chromium.launch();
const newPage = await newBrowser.newPage();
// Restore the session - already logged in!
await session.restore(newPage);

CLI Interface

PSP includes a comprehensive command-line interface for managing sessions without writing code.

Basic Commands

psp list - List all sessions
psp create - Create new session
psp launch - Launch browser
psp export - Export session data

Advanced Features

psp ui - Launch web interface
psp demo - Interactive demo
psp test - Platform compatibility
psp mcp - Start MCP server
CLI Examples
# Create a new session interactively
psp create "My Session" "Description"
# List sessions with filtering
psp list --status active --tags gmail
# Launch web interface
psp ui --port 3000
# Test platform compatibility
psp test --platform Gmail

Web Interface

PSP includes a beautiful React-based web interface for visual session management.

Launch Web Interface

Start the web interface with a single command:

psp ui

Interface will be available at http://localhost:3000

Features

  • Visual session browser
  • Real-time monitoring
  • Drag & drop organization
  • Export/import capabilities

Dashboard

  • Session statistics
  • Platform usage analytics
  • Success rate monitoring
  • Quick actions panel
