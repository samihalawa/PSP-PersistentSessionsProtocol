# Suggested Commands

## Build Commands
```bash
npm run build              # Build all packages
npm run build:types        # Build types package
npm run build:adapters     # Build adapters package
npm run build:server       # Build server package
npm run build:cli          # Build CLI package
npm run build:mcp          # Build MCP server
```

## Run Commands
```bash
npm run start:server       # Start PSP server
npm run start:cli          # Start CLI
npm run start:mcp          # Start MCP server
```

## Development
```bash
npm install                # Install dependencies
npm run build --workspaces # Build all workspaces
```

## System Commands (macOS/Darwin)
```bash
git status                 # Check git status
git log --oneline -10      # Recent commits
ls -la                     # List files
find . -name "*.ts"        # Find TypeScript files
grep -r "pattern" .        # Search in files
```
